# Hisab Desktop Application Requirements
# This application uses only Python standard library components

# Core requirements (all included in Python standard library):
# - tkinter (GUI framework)
# - re (regular expressions)
# - math (mathematical functions)

# No external dependencies required!
# The application should work with any Python 3.6+ installation.

# To install Python (if not already installed):
# - Windows: Download from python.org
# - macOS: brew install python3 or download from python.org  
# - Linux: sudo apt install python3 python3-tk (Ubuntu/Debian)
#          sudo yum install python3 tkinter (CentOS/RHEL)
